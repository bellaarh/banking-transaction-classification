# banking-transaction-classification
This project builds a machine learning model to predict customer transaction categories based on historical bank data.

## About This Project
This project was developed as part of my final submission for the Dicoding x DBS Data Science course.

## Key Project Files
- `[Klasifikasi]_Submission_Akhir_BMLP_Nabila_Rahmatul_Husna.ipynb`: Main notebook for classification modeling.
- `decision_tree_model.h5`: Decision Tree model.
- `explore__RandomForest__classification.h5`: Initial Random Forest model.
- `tuning_classification.h5`: Optimized model after hyperparameter tuning.

## Key Insights
- Compared performance between Decision Tree and Random Forest models.
- Achieved 99% accuracy after hyperparameter tuning.
- Identified key features that significantly influence transaction classification.
